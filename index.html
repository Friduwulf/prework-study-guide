HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user

CSS

  • A margin indicates how much space we want around thew outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A period preceding a word (ex. card) designates it as a class selector.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commit all files in the working branch
  • git commit -m "insert text here": associate a message to the commit
  • git pull origin main: pulls the main branch into the feature branch, hopefully see "already up to date"
  • git push origin feature/branch-name: pushes all changes to repository

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • In JavaScript, arrays are zero indexed, meaning the first value is indexed 0 instead of 1.
  • A function must be called for it to work. Simply creating a function does not execute it.
